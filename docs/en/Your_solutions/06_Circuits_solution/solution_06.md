# Task 06 – Kirchhoff's Laws again

## Problem Statement

Calculate the current flowing through the ammeter in the circuit (image-k2.png).

## Theory

By applying KVL to the loops of the bridge or multi-loop circuit, we can determine the current in any specific branch containing an ammeter.

## Step-by-Step Solution

1. **Mesh Analysis:** Define loop currents $i_a$ and $i_b$.
2. **Setup Equations:** Sum voltages around each closed path.
3. **Solve for $I_{ammeter}$:** Usually, this is the current in the central or outer branch where the instrument is placed.
